## 概要

Udemy の教材 [【0 から始める Pytest 超基礎講座】Python のプログラムを効率的にテスト](https://www.udemy.com/course/python_pytest/) の学習内容を配置するためのディレクトリです。

## Section 2

### 実行方法

- `test_` で始まるファイルがテストファイルとして認識される
- `test_` で始まる関数がテスト関数として認識される
- `pytest`: カレントディレクトリ以下のテストをすべて実行できる
- `pytest <ファイル名>`: 特定のテストファイルを実行できる
- `pytest -s`: テスト中の print 文が表示される
- `pytest -v`: 詳細な実行結果が表示される

### assert 文

- `assert <条件>`: 条件が False の場合にテストが失敗する

### クラスのテスト

- テストクラス名は `Test` で始める
- テストメソッド名は `test_` で始める

# 環境構築

## UV

参考: [Working on projects](https://docs.astral.sh/uv/guides/projects/)

### プロジェクト作成

```bash
uv init python-playground
```

### コード実行

```bash
uv run main.py
```

### 実行結果

venv が作成された

```bash
$ uv run main.py
Using CPython 3.13.3
Creating virtual environment at: .venv
Hello from python-playground!
```
